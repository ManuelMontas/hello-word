El objetivo de las paginas web

El objetivo de una pagina web es interactuar con el visitante, es decir, que los invite a realizar una accion en lo especifico. Las acciones mas comunes en un sitio web son: realizar una compra, enviar formulario o solicitud de presupuesto

Algunos objetivos pueden ser:

Comerciales:

Estan creados para promocionar los negocios de una empresa. Su finalidad es economica.


Educativo:

Permiten ver o descargar contenidos de asignaturas o temas para estudiantes o profesores.


Comunidad virtual:

Un sitio donde las personas con intereses similares se comunican con otros, normalmente por chat o foros.
