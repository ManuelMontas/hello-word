Que es una pagina web

se conoce como pagina web a un documento digital de caracter multimediatico [es decir, capaz de incluir audio, video, texto y sus combinaciones] adaptado a los estandares de la world wide web www, y a la que se puede acceder a traves de un navegador web y una conexion activa a internet, se trata del formato basico de contenidos en la red.

Una pagina web es en esencia una tarjeta de presentacion digital, ya sea para empresa, organizaciones, o personas, asi como una manera de comunicar ideaws, pensamientos, conocimientos, informaciones o teorias.

Para que siven las paginas web

Un sitio web puede ser creado para que cumpladiversos propositos como:


Medio de informacion: una pagina web puede utilizarse para compartir todo tipo de informacion, desde noticias periodica, temas especializados e incluso informacion relacionada a conceptos especificos como ideologias politicas, religiosa e incluso informacion academica.


Entretenimiento: este es uno de los usos mas comunes de las pagina web. En internet hay infinidad de sitios web que ofrecen metodos de entretenimiento que van desde plataformas de videos hasta sitios donde puedes jugar videojuegos en linea.


Educacion: uno de los grandes beneficios que trajo consigo la internet es el facil acceso al conocimiento. Hoy existen muchas plataformas que ofrecen cursos, capacidades y tutoriales que permiten adquirir conocimientos desde cualquier lugar en el que se tenga accesos a internet.
